# Visão Geral do Projeto

A Store 1 é uma empresa de comércio eletrônico que começou recentemente a coletar dados de seus clientes. O objetivo deste projeto é avaliar a qualidade de uma amostra de dados e prepará-la para análises futuras.

# Objetivo

Avaliar a qualidade dos dados, identificando inconsistências, valores ausentes e possíveis erros, garantindo que o conjunto de dados seja confiável e adequado para análise.
